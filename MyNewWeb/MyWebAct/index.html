  • CRUD

    Through a login and a previous registration you will be able to enter the CRUD application. This CRUD application will allow you to create a database with a table and predetermined fields. In this you will be able to create, modify, read and delete records from the database used (SQLite3).     How to built

    1. Download the source folder
    2. Install units (python and pillow)
    3. Open main.py in Visual Studio Code (or other)
    4. Run (In Visual Studio Code -> ctrl + f5)

    In this project, I sought to learn how to make a countdown with the Date class in Javascript.

    In this project I made the game of crossing the street but I adapted it to my style with cars, trucks, tanks, bullets and a bridge that is randomly generated with a specific time.

    In this project I made the famous memory game, in which you try to find the double cards to win.


  • Rock, Paper or Scissors

    In this project I made the game Whac - a - mole in which you must make as many clicks each time the rat appears to score points. You have 60 SECONDS!
